# data

Test data files used by the test suite. These files provide structured input for validating PyHydrate's parsing and access patterns.

## Files

- **`basic-dict-get.json`** - JSON test fixture for dictionary access pattern tests (`dict_get_tests.py`).
- **`basic-list-get.json`** - JSON test fixture for list/array access pattern tests (`list_get_tests.py`).
- **`init-test-data.json`** - JSON test fixture for initialization tests validating JSON string and file loading.
- **`init-test-data.yaml`** - YAML test fixture for initialization tests validating YAML string and file loading.
- **`init-test-data.toml`** - TOML test fixture for initialization tests validating TOML string and file loading.
